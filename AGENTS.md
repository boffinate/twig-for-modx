# Extra Repository Notes

This directory is its own Git repository.

- If you change files under `extras/twig-extra/`, commit those changes in this repository, not in the parent `modx3-twig` repository.
- Use `git` commands from `extras/twig-extra/` when staging, committing, or checking status for extra-specific work.
- Only commit in the parent repository if you intentionally changed files outside this extra.
